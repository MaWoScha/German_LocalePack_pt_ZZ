Nota importante sobre Magento Versão 1.9 / Nota importante acerca Magento Versión 1.9 / Important note on Magento Version 1.9 / Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. tem com sua própria versão 1.9, traduzido semi-automaticamente, pacote de idiomas pré-instalados. Quem quiser usar o LocalePack_pt_ZZ, você deve primeiro remover a pasta app/locale/pt_ZZ para o pacote de idiomas pode ser instalado.

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_pt_ZZ, primero debe eliminar la carpeta app/locale/pt_ZZ para que el paquete de idioma se puede instalar.

Magento Inc. has pre-installed with the 1.9 private, semi-automatically translated, language pack. Who wants to use the LocalePack_pt_ZZ must first delete the app/locale/pt_ZZ folder so that this language pack can be installed.

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_pt_ZZ benutzen möchte, muss vorher den Ordner app/locale/pt_ZZ löschen, damit dieses Sprachpaket installiert werden kann.
